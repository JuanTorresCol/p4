# Práctica 3 - Página web estática

Proyecto de ejemplo para la práctica de HTML + CSS sobre un carrito de compra.

## Estructura

- `index.html` → portada
- `productos.html` → catálogo y tabla comparativa
- `carrito.html` → resumen del carrito
- `checkout.html` → formulario de compra
- `styles.css` → estilos compartidos
- `script.js` → interacciones visuales mínimas
- `assets/` → imágenes SVG y audio local

## Despliegue en GitHub Pages

Sube el contenido al repositorio `p4` y publica desde la rama `main` usando la carpeta raíz.
