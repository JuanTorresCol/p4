Práctica 3 · Web estática

Una tienda online sencilla, limpia y con una misión.

Esta demo representa una tienda con un carrito de compra hardcodeado. Tiene cuatro páginas enlazadas, diseño uniforme con CSS y varios elementos HTML: títulos, párrafos, tablas, multimedia y formularios.

Ver catálogo Ir al carrito

Catálogo claro

Tarjetas visuales con productos destacados, precios visibles y navegación rápida entre páginas.

Carrito estático

Los artículos, cantidades y precios están definidos en el código para centrarse en HTML y CSS.

Diseño adaptable

La web se ajusta a móvil, tablet y escritorio con una maquetación simple y consistente.

Qué incluye esta práctica

  • 4 páginas HTML interconectadas.
  • Uso de etiquetas: header, nav, main, section, article y footer.
  • Tabla comparativa de productos y tabla de resumen del carrito.
  • Multimedia con imagen, vídeo embebido y audio.
  • Formulario de checkout con validación básica HTML.
